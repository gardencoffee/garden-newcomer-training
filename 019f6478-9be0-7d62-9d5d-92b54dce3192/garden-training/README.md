# Garden Newcomer Training

Веб-версия вводного тренинга для новичков Garden. Макет сделан преимущественно для desktop: ноутбук, проектор, большой экран или демонстрация в переговорной.

## Что внутри

- `index.html` — готовая страница-презентация.
- `assets/` — логотип и изображения, использованные в тренинге.

## Важно

Эта папка не предназначена для публикации в репозиторий `gardencoffee/Promo`.
Для GitHub Pages нужно создать отдельный репозиторий, например:

```text
garden-newcomer-training
```

## Как опубликовать через GitHub Pages

1. Создать новый репозиторий на GitHub.
2. Загрузить туда содержимое этой папки: `index.html`, `README.md`, `assets/`.
3. Открыть Settings -> Pages.
4. Выбрать Deploy from a branch.
5. Branch: `main`.
6. Folder: `/root`.
7. Сохранить.

После этого GitHub выдаст публичную ссылку на тренинг.

## Локальный запуск

Файл `index.html` можно открыть в браузере напрямую.

Управление:

- стрелки влево/вправо — переход между слайдами;
- кнопка "Показать заметки ведущего" — включает подсказки для тренера;
- "Печать / PDF" — подготовка версии для сохранения в PDF.
